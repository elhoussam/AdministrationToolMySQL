Introduction

Cette application est un projet fin d'étude faculter Sience Exact et science de la vie
Classe , informatique Biskra 3 LMD et cette application orientée vers les gents , qui veulent administration MS sql server 2014

Presentation

Partie 1

Cette partie nommer HEADER contien a droite le nom de login et la date ,
Cliquer sur le button droite afficher quelque option (Reconnect , changer theme etc..)

Partie 2

Cette partie nommer MENU contien tous les fonctionnalites de cette Application

Partie 3

Est une arbre pour les base des donneés et leurs table
pour voir les tables de base cliquer sur le nom de base et puis sur [+]
Partie 4

Complementaire contien des information , et les requettes , et les erreurs

Comment Utiliser

Ce titre en savoir l'utilisation de cette application

Base de donneé

  1. Creation

    2. Ecrire le nom base de donneé dans le premier champ
    [optionnel option]
    Principal : le nom de fichier de base de donneé
    Secondaire : le nom de fichier il contien des informations sur cette base de donneé

  2. Modifier

    3. Selectionner base de donneé
    Principal : renommer le fichier et leur place
    Clicker sur button information pour voir les informations sur cette base de donneé

  3. Supprimer

    4. Dans image de Creation base de donneé
    Partie Arbre Clicker a droite et cliquer sur Suppriemr

Table

  1. Creation

    2. Entrer le nom de la table sans espace , et choiser le base de donneé apparu et entrer le colonne et leurs details

  2. Ajouter Colonne

    3. Choiser la base de donneé et table et ajouter les colonnes

  3. Supprimer ou Renommer Colonne

    4. Choiser la base de donneé , table et le colonne
    pour renommer entrer le nouveau nom
    partie droite dans arbre permet de supprimer table ou renpmmer , Cliquer a droite

  4. Modifier Colonne

    5. Choiser la base de donneé , table et le colonne
    et puis modifier detail de colonne

Schema

  2. Supprimer
